Teoría de Funciones Lineales

¿Qué es una Función Lineal?

Una función lineal es una relación matemática entre dos variables donde la variable dependiente (y) cambia de manera constante con respecto a la variable independiente (x). Se representa mediante la ecuación:

\[y = ax + b\]

Donde:

  • a: Es la pendiente de la recta (tasa de cambio constante)
  • b: Es el punto de intersección con el eje Y (ordenada al origen)
  • x: Es la variable independiente
  • y: Es la variable dependiente

Elementos Fundamentales

1. La Pendiente (a)

La pendiente representa la inclinación de la recta y se puede interpretar como:

\[a = \frac{\Delta y}{\Delta x} = \frac{y_2 - y_1}{x_2 - x_1}\]
  • Pendiente Positiva (a > 0): La función es creciente
  • Pendiente Negativa (a < 0): La función es decreciente
  • Pendiente Cero (a = 0): La función es constante (paralela al eje X)
  • Pendiente Indefinida: La recta es vertical (paralela al eje Y, no es función)

2. Intersección con el eje Y (b)

El valor de b indica:

  • El punto donde la recta corta al eje Y
  • El valor de y cuando x = 0
  • La altura inicial de la función

Propiedades Importantes

1. Dominio y Rango

  • Dominio: ℝ (todos los números reales)
  • Rango: ℝ (todos los números reales)
  • Justificación: La función lineal es continua y no tiene restricciones

2. Continuidad

  • Es continua en todo su dominio
  • No tiene saltos ni interrupciones
  • Se puede trazar sin levantar el lápiz del papel

3. Monotonía

  • Estrictamente Creciente: Si a > 0
  • Estrictamente Decreciente: Si a < 0
  • Constante: Si a = 0

Formas de Representación

1. Forma Pendiente-Intersección

\[y = ax + b\]

La forma más común y útil para graficar

2. Forma Punto-Pendiente

\[y - y_1 = a(x - x_1)\]

Útil cuando conocemos un punto y la pendiente

3. Forma General

\[Ax + By + C = 0\]

Donde A y B no pueden ser simultáneamente cero

Aplicaciones Prácticas

1. Economía

  • Costos fijos y variables
  • Funciones de oferta y demanda
  • Depreciación lineal

2. Física

  • Movimiento rectilíneo uniforme
  • Relación entre voltaje y corriente (Ley de Ohm)
  • Conversiones de temperatura

3. Geometría

  • Perímetro en función del lado
  • Área de figuras simples
  • Relaciones entre ángulos

Casos Especiales

1. Recta que pasa por el origen

\[y = ax\]

Cuando b = 0, la recta pasa por el origen (0,0)

2. Recta horizontal

\[y = b\]

Cuando a = 0, la recta es paralela al eje X

3. Recta vertical

\[x = k\]

No es una función, pero es una recta en el plano

Métodos de Resolución

1. Encontrar la pendiente

  • Entre dos puntos: a = (y₂ - y₁)/(x₂ - x₁)
  • Por el ángulo: a = tan(θ)
  • Por incrementos: a = Δy/Δx

2. Encontrar la intersección

  • Con el eje Y: el valor de b
  • Con el eje X: resolver y = 0
  • Entre dos rectas: resolver el sistema de ecuaciones

Propiedades Algebraicas

1. Composición de Funciones

La composición de dos funciones lineales es también una función lineal:

\[(f \circ g)(x) = f(g(x)) = a_1(a_2x + b_2) + b_1 = (a_1a_2)x + (a_1b_2 + b_1)\]

2. Inversa de una Función Lineal

La inversa de una función lineal f(x) = ax + b es:

\[f^{-1}(x) = \frac{x - b}{a}\]

Donde a ≠ 0

3. Sistemas de Ecuaciones Lineales

Un sistema de ecuaciones lineales puede tener:

  • Una única solución (rectas secantes)
  • Infinitas soluciones (rectas coincidentes)
  • Sin solución (rectas paralelas)

Interpretación Geométrica

1. Ángulo de Inclinación

El ángulo θ que forma la recta con el eje X positivo está relacionado con la pendiente:

\[\theta = \arctan(a)\]

2. Distancia entre Puntos

La distancia entre dos puntos (x₁, y₁) y (x₂, y₂) en una recta es:

\[d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}\]

3. Paralelismo y Perpendicularidad

  • Dos rectas son paralelas si tienen la misma pendiente
  • Dos rectas son perpendiculares si el producto de sus pendientes es -1

Límites y Continuidad

1. Límites

Para cualquier función lineal f(x) = ax + b:

\[\lim_{x \to c} f(x) = f(c)\]

Donde c es cualquier número real

2. Continuidad

  • La función es continua en todo su dominio
  • No presenta discontinuidades ni saltos
  • Es diferenciable en todo punto
